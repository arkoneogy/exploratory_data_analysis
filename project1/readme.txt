Summary
--------------------------------------
This folder contains code and output for the course project -1 submission in Exploratory Data Analysis MOOC from Coursera.

Contents
--------------------------------------
runme.R					-- top level script for running all the code and producing all the plots
plot1.R 				-- function to make first plot
plot2.R 				-- function to make second plot
plot3.R 				-- function to make third plot
plot4.R 				-- function to make fourth plot

plot1.png				-- outplot plot #1, reproducible by code
plot2.png				-- outplot plot #2, reproducible by code
plot3.png				-- outplot plot #3, reproducible by code
plot4.png				-- outplot plot #4, reproducible by code

Running instructions
--------------------------------------
The code requires the dataset (names "household_power_consumption.txt") to be present in the working directory.

1. Place the codes (runme.R, plot*.R) in the same directory as the dataset
2. Execute the top level script as:
		"Rscript runme.R"

Plots will get produced in the working directory.
